How Does it Work?



Step 1

The network takes in two inputs, each between -1 and 1. These are decimals expressed as percentages that represent information about a company. These express different characteristics of company growth. Based on test data the network will be able to predict an investment success chance.


Step 2

The network has three layers, input, hidden, and output. The hidden value uses weights and biases to make an algorithm that outputs a binary value, 0 or 1, success or failure. The test values have values that are known to correlate to an output value.


Step 3

The neural network iterates through all of these values a set number of times. The user, based on their presences and insight knowledge on their investment, can choose to execute a sigmoid function or a hyperbolic function to execute their code, based on what they think is more efficient.



Equations and Calculations


The algorithm weights and biases are initially set to a random value. The hidden layer uses these values to calculate an output. The value of the first two numbers is determined to be a 1 or 0. If the output doesnâ€™t match the known result, then a backward propagation technique is started.


The backward propagation technique uses a bit of calculus to calculate the factor by how much to change the weights and the bias by. This is done with the margin of error which can be calculated in a similar manner as seen above. So after the program encounters a value that outputs an incorrect value, it will initiate backward propagation which will then alter the algorithm to accommodate for this next value.


The network will then keep iterating through all 250 values a set number of times so that it creates an efficient algorithm that accommodates for the most values as possible. In the final iteration of the 250 test data (which has been completely randomized), the program will count how many times the network was successful. This will give the success percentage to the user. The two user input values will then be plugged into the algorithm, which will result in a 1 or 0. Based on this data, the user can decide whether or not to make an investment in the company of his or her choice.


The HTML to Java is transferred through JSP. This is what transfers the data between the user input and the program as well as the network output and the display on the website. No machine libraries were used, no APIs, and the code was done manually.
